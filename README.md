# Resolucao-Refinaria-Delta9
Nessa atividade eu resolvi uma situação critica em que uma empresa fictícia onde se encontra uma Refinaria. Com essa resolução e criação dessas funções, a empresa pode voltar a utilizar e se situar cada detalhe e cada mudança que está acontecendo dentro do sistema. Com um código feito, a empresa ira operar tranquilamente.
